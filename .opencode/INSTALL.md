# Ironclad — OpenCode Installation

## Quick Install

```bash
git clone https://github.com/YOUR_USERNAME/ironclad.git
cp -r ironclad/skills/ ~/your-workspace/skills/
```

## Verification

Start a new session and request a feature. The agent should invoke skills automatically.

## Updating

```bash
cd ironclad && git pull
cp -r skills/ ~/your-workspace/skills/
```
